таблица config представляет собой настройки сайта и его расширений.
Таблица имеет следующий формат:
    variable_id - уникальный идентификатор переменной
    variable_name - имя переменной
    variable_value - значение переменной. значение может быть продиктовано
типом переменной
#TODO 01:
    variable_type - тип переменной. возможные типы переменной:
string, int, bool, float, enum
тип переменной позволяет создавать предстваление переменной
    variable_limits - определяет ограничения значений переменной. возможны
следующие варианты записи для типов:
    string - без ограничений
    bool - без ограничений
    int - 'a-b' означает, что переменная может принимать значения от а до b
включительно
    float - 'a-b' означает, что переменная может принимать значения от а до b 
включительно
    enum - 'value1:value2:value3' означает, что переменная может принимать 
одно из значений value1, value2 или value3. Возможные значения разделяются 
друг от друга двоеточием
