структура модуля - директория <MODULE_NAME> содержит 
    файл <MODULE_NAME>.class.php
    директорию templates
    файл install.sql
    файл uninstall.sql

модули представляют собой классы, имеющие атрибутами
    name - значение совпадает с названием класса
    version - обязательно использование даты релиза, yyyymmdd
операции:
    конструктор
    функции, совпадающие по названию с хуками
    process_<query> - функция, обрабатывающая запрос страницы, $_GET["p"]

описание таблицы модулей modules_tbl:
    module_id - идентификатор модуля
    module_name - имя модуля
    module_version - версия модуля. позволяет проверять обновление модуля
    module_enabled - переключатель, позволяющий отключать использование модуля
